# LLM-Powered Quantum Learning & Research Assistant

## Overview
This project aims to build an AI-powered tutor to help students understand
fundamentals of Quantum Computing. The system uses Retrieval-Augmented
Generation (RAG) to answer questions strictly from trusted quantum computing
resources.

## Objectives
- Assist beginners in learning Quantum Computing concepts
- Reduce hallucinations using document-grounded answers
- Enable explainable and research-friendly AI tutoring

## Key Features
- Document-based question answering (RAG)
- Vector database for semantic retrieval
- Modular and extensible architecture
- Dockerized deployment

## Planned Tech Stack
- Python
- FAISS (Vector Database)
- LLM (API-based or local)
- FastAPI
- Streamlit
- Docker
